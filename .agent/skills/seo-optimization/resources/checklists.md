# SEO & AI Search Checklists

Use these checklists during development to ensure MediSync meets the highest standards for search and AI visibility.

## 1. New Page Creation Checklist
- [ ] **Title Tag**: Is there a unique, keyword-rich `<title>`?
- [ ] **Meta Description**: Is there a compelling `<meta name="description">` (150-160 chars)?
- [ ] **Canonical Tag**: Does the page point to its preferred URL?
- [ ] **H1 Heading**: Is there exactly one `<h1>` that matches the search intent?
- [ ] **Heading Hierarchy**: Are `<h2>` and `<h3>` tags used logically to structure the page?
- [ ] **Image Alt Text**: Do all images have descriptive `alt` attributes?
- [ ] **JSON-LD**: Has the appropriate Schema markup been added?

## 2. AI & GEO Content Checklist
- [ ] **Direct Answer**: Does the first paragraph directly answer the primary question of the page?
- [ ] **Structure**: Are bullet points used for key features or lists?
- [ ] **Claim Evidence**: Are data points and facts cited with sources or expert quotes?
- [ ] **Topical Authority**: Does the content use industry-standard terms like "Healthcare BI", "HIMS", "PII Protection"?
- [ ] **Scannability**: Can an AI model (or human) understand the page essence in 5 seconds?

## 3. Technical SEO Audit Checklist
- [ ] **Lighthouse Score**: Does the SEO score exceed 90?
- [ ] **Mobile Friendly**: Is the layout fully responsive and touch-friendly?
- [ ] **Crawl Depth**: Is the page linked from the main navigation or a high-level page?
- [ ] **Robots Exclusion**: Is the page accidentally blocked by `robots.txt` or `noindex`?
- [ ] **Page Speed**: Does the page load in under 2 seconds on a 3G connection?
